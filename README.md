# Keen Keeper

Keen Keeper is a responsive friendship tracker built with React, TypeScript, React Router, Tailwind CSS, and Recharts.

## Description

This app helps you keep track of friends, contact deadlines, and recent check-ins. It includes a home dashboard, friend detail pages, a timeline of interactions, and analytics for call/text/video patterns.

## Technologies

- React
- TypeScript
- Vite
- React Router DOM
- Tailwind CSS
- Recharts
- react-hot-toast
- Lucide React icons

## Features

- Responsive navbar with active page highlighting
- Home dashboard with banner, summary cards, and friend cards
- Friend details page with status, tags, bio, and quick check-in actions
- Timeline page with filter controls
- Analytics page with a pie chart showing interaction counts
- Toast notifications for check-in actions
- 404 page for unknown routes

## Available Scripts

- `npm install` — install dependencies
- `npm run dev` — start the development server
- `npm run build` — create a production build
- `npm run preview` — preview the production build
